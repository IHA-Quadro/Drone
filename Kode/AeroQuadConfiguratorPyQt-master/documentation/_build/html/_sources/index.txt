Welcome to the AeroQuad Configurator Developer Guide's documentation!
=====================================================================

This is the developer's guide for the Python based AeroQuad Configurator.  The Configurator is a visual tool used to verify the correct setup and operation of your AeroQuad flight control board.  It can also upload the latest firmware to take advantage of the latest flight control features.

This developer's guide will show you how to add your own features and capabilities to the AeroQuad Configurator.

If you are looking for a User's Guide for the Configurator, `please check out our wiki! <https://github.com/AeroQuad/AeroQuadConfiguratorPyQt>`_
